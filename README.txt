
AI-Based Personalized Diet Planner
----------------------------------

Description:
This web-based intelligent application provides customized meal plans 
based on user inputs like age, gender, height, weight, diet type, and health goal.

Features:
- Tailored meal plans for weight loss, muscle gain, healthy skin, hair growth, and acne reduction.
- Displays breakfast, lunch, dinner, and food to avoid.
- Modern UI with gradient themes and professional layout.

Tech Stack:
- Flask (Backend)
- HTML, CSS (Frontend)
- JSON (Data Storage)

Run Instructions:
1. pip install flask
2. python app.py
3. Open http://127.0.0.1:5000
